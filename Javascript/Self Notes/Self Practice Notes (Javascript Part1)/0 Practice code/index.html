Learning about Prototypes

function provides an free space called prototype object

__proto__ or [[prototype]]

__proto__ or [[prototype]] both means the same, Note:the prototype is completely different than __proto__ or [[prototype]]

__proto__ or [[prototype]] every object have an proto, a proto stores the reference of another object, means when an object does not have the key value required by the user then it will search in the proto

In ecma script documentation the __proto__ is represented as [[prototype]] but both are same

New keyword

The three work which is done by New Keyword:

  1. make an this object inside the function which is called using a new keyword
  2. return this object
  3. link the __proto__ implicitly to this object , So we do not need to do it explicitly

Constructor Function

Whenever we are creating a constructor function, it is a convensions that the name of the function starts with an capital letter

So it benifits are colleagues to understand that, this is a constructor function, it should need to call using new keyword
